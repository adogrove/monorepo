# Installation

The `@adogrove/adonis-geolite2` package is available on [npm](https://www.npmjs.com/package/@adogrove/adonis-geolite2).
You can install it using the following ace command to automagically configure it:
```
node ace add @adogrove/adonis-geolite2
```

Alternatively, you can install it manually.
::: code-group
```sh [npm]
npm install @adogrove/adonis-geolite2
node ace configure @adogrove/adonis-geolite2
```
```sh [pnpm]
pnpm install @adogrove/adonis-geolite2
node ace configure @adogrove/adonis-geolite2
```
```sh [yarn]
yarn add @adogrove/adonis-geolite2
node ace configure @adogrove/adonis-geolite2
```
:::
